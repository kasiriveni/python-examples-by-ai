## Core Example
This example brings together variables, collections, a function, and a loop.

```python
numbers = [1, 2, 3, 4]
profile = {"name": "Bob", "active": True}

def double(value):
	return value * 2

for number in numbers:
	print(profile["name"], double(number))
```
