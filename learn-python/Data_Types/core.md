# Core Python Concepts

- Work with numbers, strings, booleans, and `None` as the language's basic value types.
- Use lists, tuples, sets, and dictionaries to model grouped data.
- Understand mutability, immutability, and how values are passed around in Python.
- Convert between types intentionally with constructors such as `int()`, `str()`, and `list()`.
- Use bytes, enums, and datetime-related types when domain data needs them.
- Know when a type's behavior affects hashing, equality, ordering, or serialization.
