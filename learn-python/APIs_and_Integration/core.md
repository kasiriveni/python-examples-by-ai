# Core Python Concepts

- Call external APIs with libraries such as `requests` or `httpx`.
- Handle authentication headers, tokens, API keys, and signed requests safely.
- Parse JSON responses and map external data into Python structures.
- Add retries, timeouts, and error handling for unreliable network calls.
- Work with pagination, rate limits, and backoff strategies.
- Keep integration code isolated so external dependencies do not leak across the codebase.
