# Core Python Concepts

- Use SQL and database drivers to create, read, update, and delete persistent data.
- Manage connections, transactions, and cursors so database state stays consistent.
- Use ORMs when they improve productivity, but still understand the SQL underneath.
- Work with relational and NoSQL databases based on data shape and access patterns.
- Parameterize queries and validate input to avoid injection issues.
- Think about indexing, query cost, and schema design as part of application performance.
