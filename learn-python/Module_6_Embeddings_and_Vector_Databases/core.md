# Core Python Concepts

- Convert text or other inputs into embeddings for similarity search.
- Chunk documents so retrieval balances recall, context, and cost.
- Use cosine similarity or related measures to compare vector representations.
- Store and query embeddings in vector databases such as FAISS, Chroma, or Pinecone.
- Keep metadata alongside vectors so filtering and attribution remain possible.
- Treat indexing, updates, and retrieval quality as part of system design.
