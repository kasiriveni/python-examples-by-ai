# Core Python Concepts

- Open files with the correct mode for reading, writing, appending, or binary access.
- Use `with` blocks so files always close correctly.
- Work with text, CSV, JSON, and binary data based on the file format.
- Use `pathlib` for cleaner path manipulation across operating systems.
- Stream large files instead of loading everything into memory at once.
- Handle encodings and missing files explicitly to avoid fragile scripts.
