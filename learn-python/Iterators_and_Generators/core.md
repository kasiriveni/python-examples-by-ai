# Core Python Concepts

- Implement the iterator protocol with `__iter__` and `__next__` when custom iteration is needed.
- Use generator functions with `yield` for lazy, memory-efficient sequences.
- Prefer generator expressions for simple streaming transformations.
- Use `itertools` to compose iteration pipelines from reusable building blocks.
- Understand one-pass consumption and when iterators cannot be reused.
- Use `send`, `throw`, and `close` only when a generator's control flow truly needs them.
