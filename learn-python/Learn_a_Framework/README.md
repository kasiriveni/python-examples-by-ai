# Learn a Framework

This folder contains examples for popular Python frameworks:

1. **Django**: Batteries-included web framework.
2. **Flask**: Lightweight micro-framework.
3. **FastAPI**: Modern, async, API-first framework.
4. **Tornado**: Async web framework.
5. **Sanic**: Async web framework with high performance.
6. **aiohttp**: Async web framework for building web servers.
