# Core Python Concepts

- Break problems into small steps and write simple, testable functions.
- Practice variables, input/output, arithmetic, and string formatting.
- Use conditionals and loops to control program flow.
- Work with common data structures such as lists, tuples, sets, and dictionaries.
- Apply indexing, slicing, and basic transformations to strings and collections.
- Debug small scripts by printing intermediate values and checking assumptions.
