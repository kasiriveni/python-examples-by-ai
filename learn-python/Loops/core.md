# Core Python Concepts

- Use `for` loops to iterate over sequences and other iterables.
- Use `while` loops when repetition depends on a changing condition.
- Combine `range`, `enumerate`, and `zip` to write cleaner iteration logic.
- Control execution with `break`, `continue`, and loop `else` when it adds clarity.
- Avoid deeply nested loops when data can be transformed more directly.
- Recognize when a comprehension is clearer than an explicit loop.
