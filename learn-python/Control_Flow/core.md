# Core Python Concepts

- Use `if`, `elif`, and `else` to branch based on conditions.
- Combine boolean expressions with `and`, `or`, and `not` carefully.
- Use `match` and `case` when structural pattern matching improves readability.
- Control loop execution with `break`, `continue`, and `pass`.
- Prefer guard clauses and clear branching over deeply nested logic.
- Write conditions that are explicit and easy to test.
