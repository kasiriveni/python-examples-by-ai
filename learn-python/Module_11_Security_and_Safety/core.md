# Core Python Concepts

- Protect secrets, credentials, and sensitive data throughout application workflows.
- Validate and sanitize untrusted input before using it in code or prompts.
- Defend against prompt injection, data leakage, and unsafe tool execution in AI systems.
- Apply authentication, authorization, and least-privilege principles consistently.
- Review dependencies and external integrations for avoidable security risk.
- Design systems so failures default to safer behavior instead of broader exposure.
