# Core Python Concepts

- Define functions with `def` to name reusable units of behavior.
- Use positional, keyword, default, and variadic arguments effectively.
- Return values explicitly instead of depending on hidden side effects.
- Understand local scope, global scope, and closures.
- Use recursion when a problem is naturally recursive and the depth is controlled.
- Treat functions as first-class objects that can be passed, stored, and wrapped.
