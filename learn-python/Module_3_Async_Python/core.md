# Core Python Concepts

- Use `async` and `await` to coordinate many I/O-bound operations efficiently.
- Create tasks and combine them with primitives such as `gather` and queues.
- Handle cancellation, timeouts, and backpressure in long-running async workflows.
- Use async HTTP clients, streams, and context managers correctly.
- Understand where threads or processes fit when async alone is not enough.
- Keep async code structured so control flow remains understandable under load.
