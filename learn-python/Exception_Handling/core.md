# Core Python Concepts

- Use `try`, `except`, `else`, and `finally` to manage failures deliberately.
- Raise specific exceptions when code detects invalid state or input.
- Create custom exception classes for domain-specific error handling.
- Preserve useful stack information rather than hiding the original cause.
- Clean up resources reliably even when operations fail.
- Treat exceptions as part of program design, not just as afterthoughts.
