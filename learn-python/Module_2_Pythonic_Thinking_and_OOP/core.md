# Core Python Concepts

- Use classes, inheritance, composition, and polymorphism to model real systems.
- Prefer Pythonic constructs such as comprehensions, iterators, and context managers.
- Use dataclasses, named tuples, and properties to express intent clearly.
- Understand dunder methods and how objects integrate with built-in language behavior.
- Apply decorators and higher-order functions where they simplify repeated patterns.
- Add type hints and utility functions from `functools` when they improve clarity.
