# Core Python Concepts

- Use arithmetic, comparison, logical, identity, and membership operators correctly.
- Understand assignment and augmented assignment behavior on mutable objects.
- Learn operator precedence so expressions evaluate the way you expect.
- Use bitwise operators when working with flags, masks, or low-level representations.
- Implement operator overloading through dunder methods only when it makes an API clearer.
- Prefer explicit expressions when a compact one becomes hard to read.
