# Core Python Concepts

- Use the standard library before adding third-party dependencies unnecessarily.
- Reach for `pathlib`, `os`, and `sys` for system and filesystem tasks.
- Use `collections`, `itertools`, and `functools` to write cleaner core logic.
- Handle dates, JSON, regular expressions, and subprocesses with the built-in modules designed for them.
- Treat the standard library as a broad toolbox that rewards familiarity over reinvention.
