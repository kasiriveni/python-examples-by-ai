## 8. Code Explanation Bot

Paste code and get an AI explanation, bug suggestions, and optimization ideas.

**Features**

* Code summary
* Bug suggestions
* Optimization ideas

**Skills**

* Developer tools
* Prompt engineering

---

## How to Run

```bash
pip install -r requirements.txt
python app.py
```

Open http://127.0.0.1:5000 in your browser.

Add your `OPENAI_API_KEY` to a `.env` file to enable real explanations.
Without a key the app runs in demo mode.
