## 10. Personal Learning Assistant

AI tutor for Python learning.

**Features**

* Ask coding doubts
* Practice questions
* Learning roadmap

**Skills**

* Educational AI workflows

---

## How to Run

```bash
pip install -r requirements.txt
python app.py
```

Open http://127.0.0.1:5000 in your browser.

Add your `OPENAI_API_KEY` to a `.env` file to enable real AI tutoring.
Without a key the app runs in demo mode.
