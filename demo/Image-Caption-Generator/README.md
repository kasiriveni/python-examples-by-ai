## 6. Image Caption Generator

Upload an image and get AI-generated captions.

**Features**

* Image upload
* Caption generation
* Social media caption ideas

**Skills**

* Computer vision basics
* Multimodal AI

---

## How to Run

```bash
pip install -r requirements.txt
python app.py
```

Open http://127.0.0.1:5000 in your browser.

Add your `OPENAI_API_KEY` to a `.env` file to enable real captions.
Without a key the app runs in demo mode.
