## 7. YouTube Video Summarizer

Paste a YouTube link and get a summary with key points.

**Features**

* Extract transcript
* Summary generation
* Key points

**Skills**

* APIs
* Summarization

---

## How to Run

```bash
pip install -r requirements.txt
python app.py
```

Open http://127.0.0.1:5000 in your browser.

Add your `OPENAI_API_KEY` to a `.env` file to enable real summarization.
Without a key the app runs in demo mode.
