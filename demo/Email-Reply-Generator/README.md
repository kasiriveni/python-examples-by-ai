## 4. Email Reply Generator

Generate professional email replies.

**Features**

* Input email text
* Choose tone (formal/casual)
* AI-generated response

**Skills**

* Text processing
* UI basics

---

## How to Run

```bash
pip install -r requirements.txt
python app.py
```

Open http://127.0.0.1:5000 in your browser.

Add your `OPENAI_API_KEY` to a `.env` file to enable real AI generation.
Without a key the app runs in demo mode.
