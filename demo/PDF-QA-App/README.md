## 5. PDF Question Answering App

Ask questions from PDF documents.

**Features**

* Upload PDF
* Search answers from content
* Summarize sections

**Skills**

* PDF parsing
* Embeddings
* Vector database basics

---

## How to Run

```bash
pip install -r requirements.txt
python app.py
```

Open http://127.0.0.1:5000 in your browser.

Add your `OPENAI_API_KEY` to a `.env` file to enable real AI answers.
Without a key the app runs in demo mode.
