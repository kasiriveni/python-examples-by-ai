## 9. Interview Question Generator

Generate interview questions with answers by topic and difficulty.

**Features**

* Select skill/topic
* Beginner to advanced levels
* Answers with explanation

**Skills**

* Structured prompts

---

## How to Run

```bash
pip install -r requirements.txt
python app.py
```

Open http://127.0.0.1:5000 in your browser.

Add your `OPENAI_API_KEY` to a `.env` file to enable real questions.
Without a key the app runs in demo mode.
